## Question Description

The code below is intended to count the number of times each name appears in a list. Your goal is to complete the AggregateNames function to return the correct result, i.e.

| Names | Count |
|---|---|
| Adrian | 3 |
| Alexander | 1 |

## Sample 

### Input

```
John
William
Mary
Ben
John
Ben
Mary
John
Ben
John
```

### Output

```
Ben 3
John 4
Mary 2
William 1
```
